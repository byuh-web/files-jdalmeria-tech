Let's Learn CSS Selectors!

List out the different ways to select an item.

Ways to select an item:

  • Element selector
  • Class selector
  • ID selector
  • Attribute selector
  • Pseudo-class selector
  • Pseudo-element selector

What is the difference between an ID and a Class?

An ID is a unique identifier for an HTML element, while a Class is a reusable identifier that can be applied to multiple elements.

What is the difference between a SPAN tag and a DIV tag?

A SPAN tag is an inline element used to apply styles or scripts to a portion of text, while a DIV tag is a block-level element used to group other HTML elements.

What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?

A DIV tag is a generic container for grouping elements, while HEADER, MAIN, and FOOTER are semantic elements that define the structure and meaning of the page.

What are the different ways to select a color?

Different ways to select a color include using color names (e.g., "red"), hexadecimal values (e.g., "#ff0000"), RGB values (e.g., "rgb(255, 0, 0)"), or HSL values (e.g., "hsl(0, 100%, 50%)").
